## Run Locally

Clone the project

```bash
      git clone https://github.com/Mamun1646/Crud_NestJs_restApi_MySQL.git

```

Go to the project directory

```bash
  cd Crud_NestJs_restApi_MySQL
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon index.js
```
